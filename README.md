# 실시간 멀티플레이어 빙고 게임

Socket.IO를 사용한 실시간 웹 기반 빙고 게임입니다. 여러 명이 같은 주소로 접속하여 함께 플레이할 수 있습니다.

## 🎮 게임 특징

- **실시간 멀티플레이어**: 여러 명이 동시에 접속하여 함께 플레이
- **방장 시스템**: 첫 번째 접속자가 자동으로 방장이 되어 게임을 제어
- **5x5 빙고판**: 각 플레이어에게 1~25 숫자가 랜덤 배치된 빙고판 제공
- **실시간 업데이트**: 번호 호출 시 모든 플레이어의 보드가 자동 업데이트
- **빙고 계산**: 가로, 세로, 대각선 빙고 줄 수를 자동 계산
- **시각적 피드백**: X 마킹 애니메이션 및 빙고 완료 알림

## 📋 요구사항

- Node.js (v14 이상 권장)
- npm (Node.js와 함께 설치됨)

## 🚀 설치 및 실행 방법

### 1. 의존성 설치

프로젝트 폴더에서 다음 명령어를 실행하세요:

```bash
npm install
```

이 명령어는 다음 패키지들을 설치합니다:
- `express`: 웹 서버
- `socket.io`: 실시간 통신

### 2. 서버 시작

다음 명령어로 서버를 시작합니다:

```bash
npm start
```

서버가 성공적으로 시작되면 다음과 같은 메시지가 표시됩니다:

```
서버가 http://localhost:3000 에서 실행 중입니다.
```

### 3. 게임 접속

웹 브라우저를 열고 다음 주소로 접속하세요:

```
http://localhost:3000
```

**중요**: 여러 명이 함께 플레이하려면 모든 참가자가 같은 주소로 접속해야 합니다.
- 로컬 네트워크: `http://[서버IP주소]:3000` (예: `http://192.168.0.100:3000`)
- 배포 환경: 배포된 서버 주소 사용

## 🎯 게임 플레이 방법

### 1. 입장하기

1. 접속 후 닉네임을 입력합니다
2. "입장하기" 버튼을 클릭합니다
3. 가장 먼저 입장한 사람이 자동으로 **방장**이 됩니다

### 2. 게임 시작 (방장만 가능)

1. 방장은 화면에 "방장" 배지가 표시됩니다
2. "게임 시작" 버튼을 클릭합니다
3. 모든 참가자에게 5x5 빙고판이 자동 생성됩니다

### 3. 번호 뽑기 (방장만 가능)

1. 방장이 "번호 뽑기" 버튼을 클릭합니다
2. 1~25 중 랜덤한 번호가 호출됩니다
3. 모든 참가자의 빙고판에서 해당 번호가 자동으로 X 표시됩니다

### 4. 빙고 확인

- 각 참가자의 빙고 줄 수가 실시간으로 업데이트됩니다
- 3줄 이상 달성 시 게임이 종료되고 우승자가 발표됩니다
- 다른 참가자의 보드를 클릭하면 상세 보드를 볼 수 있습니다 (X만 표시)

### 5. 게임 재시작 (방장만 가능)

1. 게임 종료 후 방장이 "다시 시작" 버튼을 클릭합니다
2. 새로운 빙고판이 생성되어 다시 플레이할 수 있습니다

## 🔧 기술 스택

- **백엔드**: Node.js + Express + Socket.IO
- **프론트엔드**: 순수 HTML + CSS + Vanilla JavaScript
- **통신**: WebSocket (Socket.IO)

## 📝 주요 기능

### 서버 기능 (server.js)

- Socket.IO를 통한 실시간 통신
- 게임 상태 관리 (플레이어, 호출된 번호, 게임 상태)
- 빙고판 생성 및 빙고 줄 수 계산
- 방장 자동 지정 및 변경
- 플레이어 입장/퇴장 처리

### 클라이언트 기능 (client.js)

- 닉네임 입력 및 입장
- 실시간 빙고판 렌더링
- 다른 플레이어 보드 미리보기
- 번호 호출 및 마킹 애니메이션
- 빙고 완료 알림 모달

## 🎨 UI 특징

- 파란색 계열 그라데이션 배경
- 큰 버튼과 명확한 텍스트 (터치 친화적)
- 실시간 업데이트 애니메이션
- 반응형 디자인 (모바일 지원)

## ⚠️ 주의사항

- 서버가 실행 중이어야 모든 클라이언트가 접속할 수 있습니다
- 방장이 나가면 자동으로 다음 플레이어가 방장이 됩니다
- 같은 닉네임은 사용할 수 없습니다
- 게임 중에는 방장만 번호를 뽑을 수 있습니다

## 🐛 문제 해결

### 서버가 시작되지 않는 경우

1. `npm install`이 제대로 실행되었는지 확인
2. 포트 3000이 이미 사용 중인지 확인
3. Node.js 버전이 14 이상인지 확인

### 클라이언트가 서버에 연결되지 않는 경우

1. 서버가 실행 중인지 확인
2. 방화벽 설정 확인
3. 같은 네트워크에 연결되어 있는지 확인 (로컬 네트워크의 경우)
